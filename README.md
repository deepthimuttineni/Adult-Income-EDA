# Adult Income Dataset EDA

Data cleaning and exploratory analysis on the **Adult Income (Census Income)** dataset from the **UCI Machine Learning Repository** (48,842 records, features including age, education, occupation, work class, hours per week, capital gain/loss, and income).

## What's in this notebook

- Checked for missing values, duplicate records, and potential outliers
- Performed exploratory data analysis with histograms, boxplots, bar charts, and a correlation heatmap
- Compared working hours across income groups
- Engineered three new features (**Age Group**, **Work Hours Category**, and **Net Capital**) to support future machine learning models
- Summarized the key insights from the analysis

## Files

- **Adult_Income_EDA.ipynb** – The notebook
- **adult.csv** – The original dataset
